# Print CV Generator — Quick Start Guide

## 🎯 Generate Your Professional CV in 3 Steps

### Step 1: Click Print Button
- Open your portfolio page in browser
- Click the **printer icon** (📱 in the floating button group)
- A new window opens with your CV preview

### Step 2: Configure Print Settings
```
Setting              Value
─────────────────────────────
Destination:         Save as PDF
Printer:             (default print as PDF)
Orientation:         Portrait
Paper size:          A4
Margins:             None
Scale:               100%
Headers/Footers:     OFF
Background graphics: ON
```

### Step 3: Save PDF
- Name: `Deboraj_Roy_CV.pdf`
- Location: Your preferred folder
- Click "Save"
- ✅ Done!

## ⚠️ Troubleshooting

### Issue: Print dialog doesn't appear
**Solution**: 
- Check browser pop-up blocker settings
- Allow pop-ups for this domain
- Refresh page and try again

### Issue: PDF looks wrong
**Solution**:
- In browser print settings, select "None" for margins
- Set scale to 100% (not 90% or 110%)
- Ensure "Background graphics" is ON
- Try regenerating

### Issue: Text is small
**Solution**:
- Check browser zoom is 100% (Ctrl+0)
- Print scale should be 100%
- Font is intentionally 11pt for readability

### Issue: Can't open PDF
**Solution**:
- Download a PDF reader (Adobe Reader, Foxit, etc.)
- Try opening with different reader
- File might not have saved completely

## 🔄 Updating Your CV

### To Update Content
1. Edit `window.CV` object in `cv-print.js`
2. Add/modify experience, projects, education, etc.
3. Click print button to generate new PDF
4. No need to modify CSS or HTML structure

### To Change Styling
1. Edit `print.css` file
2. Common changes:
   - Font size: Search `font-size:`
   - Colors: Search `color:`
   - Margins: Search `@page`
3. Regenerate PDF

### To Customize Print Behavior
1. Edit `generatePrintCV()` function in `cv-print.js`
2. Modify window size, timeout, or print settings
3. Test changes in browser print dialog

## 📞 Quick Help

### FAQ

**Q: How do I update my CV?**
A: Edit the `window.CV` object in `cv-print.js`, then regenerate PDF.

**Q: Can I change the fonts?**
A: Yes! Edit the font-family in `print.css` or `buildPrintDoc()`.

**Q: How often should I update my CV?**
A: After each significant achievement or job change (typically quarterly).

**Q: Is the PDF ATS-compatible?**
A: Yes! It uses semantic HTML and plain text only.

**Q: Can I print it instead of PDF?**
A: Yes! Select printer instead of "Save as PDF" in print dialog.

**Q: How do I share the PDF?**
A: Email attachment or upload to career websites (LinkedIn, Indeed, etc.).

**Q: Can I edit the PDF after saving?**
A: Better to update source data and regenerate. Use free tools like PDFtk if needed.

**Q: What if print dialog doesn't appear?**
A: Check browser pop-up blocker. Allow pop-ups for this domain.
